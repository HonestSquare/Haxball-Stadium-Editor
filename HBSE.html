    저장

    • 다운로드 버튼으로 맵 파일을 디바이스로 내보낼 수 있습니다. 현재 버전에서는 지원하지 않는 기능입니다.

    도구

    한 맵에 최대 255개까지의 물체를 둘 수 있습니다. 괄호 안은 단축키입니다.
    • 선택 (1)
      • 선택할 대상을 클릭하여 선택하십시오.
      • 여러 대상을 선택하려면 Shift를 누른 상태에서 클릭하십시오.
      • 여러 대상을 선택하려면 클릭한 상태에서 드래그하십시오.
      • 곡선을 그리려면 세그먼트를 클릭한 상태에서 드래그 하십시오.
      • 선택된 대상을 돌리려면 클릭하고 드래그하여 이동하십시오.
      세그먼트를 이동하려면 세그먼트를 클릭하여 선택한 다음 버텍스를 끌어다 놓으십시오.
    • 회전 (2)
      • 회전 중심을 배치하려면 클릭하십시오.
      • 대상을 회전하려면 클릭하고 드래그 하십시오.
    • 스케일 (3)
      • 스케일 중심을 배치하려면 클릭하십시오.
      • 크기를 조정하거나 변경하려면 클릭하고 드래그 하십시오.
      일부 특정 물체는 스케일 조절이 잘 되지 않습니다. 디스크와 곡선 세그먼트는 수직 또는 수평으로 스케일을 조절할 때 타원형 모양으로 두는 게 균형이 맞습니다. Haxball이 타원을 지원하지 않아서 형태를 정확하게 맞춰야 합니다.
      • 디스크는 스케일 조정했을 때 균형적인 영역을 가집니다.
      • 곡선 세그먼트는 첫 번째 점에서 동일한 접선을 유지합니다.
      • 세그먼트, 버텍스, 플레인은 완벽하게 확장됩니다.
    • 세그먼트 (4) - 세그먼트는 두 버텍스를 연결합니다. 장식이나 벽으로 쓸 수 있습니다.
      • 세그먼트를 추가하려면 클릭하고 드래그 하십시오.
      • 세그먼트를 연결하려면 두 양끝의 버텍스를 클릭하십시오.
    • 버텍스 (5) - 버텍스는 특수 속성을 가질 수 있는 중요점입니다.
      • 버텍스를 배치하려면 클릭하십시오.
    • 디스크 (6) - 디스크는 움직일 수있는 공과 플레이어를 제외한 유일한 물체입니다. 장식으로 꾸미거나 다른 물체와 충돌시킬 수 있습니다.
      • 기본값으로 설정된 크기로 디스크를 생성하려면 클릭하십시오.
      • 임의의 크기로 디스크를 생성하려면 클릭하고 드래그 하십시오.
    • 골 (7) - 골 영역에 디스크가 들어가면 해당 골의 반대팀이 득점을 얻습니다.
      • 골대를 배치하려면 클릭하고 드래그 하십시오.
      • 레드팀은 왼쪽, 블루팀은 오른쪽에 두는 게 일반적입니다.
      블루팀이 빨간색 골대에 득점을 합니다.
    • 플레인 (8) - 플레인은 길이 제한에 구애받지 않는 벽입니다.
      • 플레인을 배치하려면 클릭하십시오. 경기장 중앙을 향할 겁니다.

    속성 편집기

    • 특정 대상을 선택하면 해당 공통 속성이 왼쪽 하단의 속성 탭에 나열됩니다.
    • 여기서 하나가 바뀌면 선택된 대상 모두에 적용됩니다.
    • 새로 만들어진 오트젝트의 속성은 해당 대상에 적용됩니다.

    속성

    • 속도(speed) - 디스크의 초기 속도입니다. 기본값은 (0, 0) 입니다.
    • 바운스(bCoef) - 튕기는 정도입니다. has. Officially, this is a number between 0 and 1. Values below 0 and above 1 can also be used with care (and haxketball nets). The standard value for walls is 1. The standard value for goal nets is 0.1.
    • 충돌 마스크(cMask) - 대상이 충돌하는 그룹 목록입니다. 플레이어, 공은 "ball, red, blue, wall"의 마스크를 가지고 있습니다. 레드팀 킥오프 중에 모든 플레이어는 "redKO"라는 추가적인 충돌 마스크(cMask)를 갖습니다. 블루팀 킥오프 중에 모든 플레이어는 "blueKO"라는 추가적인 충돌 마스크(cMask)를 갖습니다.
    • 충돌 그룹(cGroup) - 대상이 속한 그룹입니다.
      • ball - 공입니다.
      • red - 레드팀 플레이어입니다.
      • blue - 블루팀 플레이어입니다.
      • wall - 벽입니다.
      • redKO - 레드팀 킥오프 벽입니다.
      • blueKO - 블루팀 킥오프 벽입니다.
      • all - 나머지 위 대상들을 모두 포괄합니다.
      • kick - 킥 강도가 적용됩니다.
      • score - 골인 판정이 적용됩니다
      • c0 - 특별한 역할은 없는 임의의 속성입니다.
      • c1 - 특별한 역할은 없는 임의의 속성입니다.
      • c2 - 특별한 역할은 없는 임의의 속성입니다.
      • c3 - 특별한 역할은 없는 임의의 속성입니다.
    • trait - 많은 물체에 공통적으로 지명된 속성 집합입니다.
    • x - x축, 가로 위치입니다.
    • y - y축, 세로 위치입니다.
    • 곡률(curve) - 세그먼트의 곡선 각도입니다. -340~340의 범위에서 수치값을 입력할 수 있습니다. 범위를 벗어난 값은 오버플로우 또는 다운플로우 되므로 적용할 수 없습니다.
    • 곡률(curveF) - 세그먼트의 곡선을 대체적으로 나타냅니다. curveF의 값이 있는 경우 curve의 값은 무시됩니다. 정밀 손실 없이 맵을 추출할 때만 유용하며, 추출된 맵을 편집할 땐 제거하고 curve만 사용하는 것이 좋습니다.
    • 시각화(vis) - 투명도 조절 옵션입니다. true 또는 false로 설정할 수 있습니다.
    • 색상(color) - 뜻 그대로 색상입니다. #이 빠진 6자리 16진수로 구성된 색상 코드가 적용됩니다.
    • normal - 플레인이 향하는 방향입니다.
    • dist - The distance of a plane from the center of the stadium. When it is positive, it keeps objects away from the center. When it is negative it keeps object close to the center.
    • 반경(radius) - 디스크의 반지름입니다.
    • 역질량(invMass) - 디스크의 역질량입니다. 값이 0이면 움직이지 않습니다. 값을 높일수록 디스크의 무게가 가벼워집니다. 값이 0 미만으로 둘 때는 주의가 필요합니다. 디스크의 기본값은 0입니다.
    • 좌표(pos) - 디스크의 중심 x, y 좌표입니다.
    • p0 - 골대 한 쪽 끝의 좌표입니다.
    • p1 - 골대 반대쪽 끝의 좌표입니다.
    • 팀(team) - 레드팀, 블루팀. 이렇게 두 팀밖에 없는데 뭘 바라세요?
    • 제동(damping) - 디스크의 감속률입니다. 1은 감속이 없으며, 기본값은 0.99입니다.

    상단 퀵툴

    상단 퀵툴을 통해 맵 테스트를 위해 게임에 바로 접속하거나 버그 제보 및 문의도 할 수 있습니다.
    • 뉴스
      • 게임 버전 업데이트 목록을 모아놓은 곳입니다.
    • 플레이
      • 게임에 바로 접속할 수 있습니다. 맵 테스트를 위해 빠른 이동이 가능합니다.
    • 정보
      • 게임 버전 업데이트 목록을 모아놓은 곳입니다.
    • 커뮤니티
      • 전세계에 있는 커뮤니티 목록입니다.
    • Headless Host
      • 봇방을 생성할 수 있습니다. 코드는 기본적으로 제공하지 않으므로 따로 코드 파일이 있어야 합니다.
    • 플레이(플래시)
      • 플래시 기반의 구버전으로 접속이 가능합니다. 단 호환성에 문제가 있을 수 있습니다.

    HaxBall은 basro가 개발한 온라인 멀티플레이 게임입니다.

    UM

    대한민국 제일의 haxball 유즈맵을 전문적으로 다루는 커뮤니티입니다.

      • UM의 역사가 이 네이버 카페에서부터 시작했습니다. 여러 강좌들이 올려야 왔으며, 교류가 가장 활발한 네이버 haxball 유즈맵 카페입니다.
    • haxball UseMap channel
      • haxball 유즈맵을 영상 매체로써 다루는 유튜브 채널입니다.
    • haxball UseMap storage
      • 국내외 자료들을 모으는 일종의 블로그&커뮤니티입니다.
